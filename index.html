d13framework

No Bells, No Whistles — Just Simplicity

A verrrrry simple PHP and MySQL content management system with just enough functionality to run a personal website, all powered by the incredible Twitter Bootstrap.

  Download now!   View on GitHub

...or clone on GitHub : $ git clone git://github.com/d13design/d13framework

Features? Not many, but we've got a few...

Pretty URLs

Create search engine friendly URLs out of the box with tailorable post 'slugs' and quick validation on new articles and pages.

Web-based admin

Simple online admin area for content management of pages, articles and sections.

Simple structure

Nothing over the top just a simple page / article / section structure to give you some flexibility.

Multiple templates

Support for multiple layout templates with a cracking default theme built on the Twitter Bootstrap framework.

Template helpers

Tons of pre-built helpers to call from directly within your own templates - anything you can see in the default theme you can create with helpers!

Template help

The admin pages are full of template help information - all the helpers and code snippets you'll need are listed here.

Multi-site support

The simple TBL_PRE config option lets you run multiple sites from the same, shared database - great!

Custom properties

Tag articles with a string of custom data that you can then read and interpret in your templates using PHP. Nothing fancy, just something simple.
